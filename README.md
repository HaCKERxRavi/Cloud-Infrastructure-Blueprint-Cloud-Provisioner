# Enterprise Cloud-Based System Provisioner (AIO Automation)

An automated DevOps orchestration engine written in Python designed to dynamically model, validate, and provision multi-tier enterprise cloud architectures, network boundaries, and compute topologies.

## 🛠️ Key Technical Capabilities
- **Identity Context Validation:** Enforces strict environmental profile mapping before infrastructure execution.
- **Network Virtualization Subnetting:** Configures segregated Virtual Private Clouds (VPC) with dedicated Public and Private Subnet topologies.
- **Compute Cluster Orchestration:** Provisions production-grade nodes mapped with distinct CPU/RAM configurations and automatic network interface attachments.
- **Dynamic Asset State Export:** Synthesizes deployment telemetry into an immutable centralized JSON infrastructure manifest for configuration tracking.

## 📁 Project Components
- `provisioner.py` - Core automation logic handling environmental execution flow.
- `infrastructure_report.json` - Automatically generated centralized infrastructure state logs.
